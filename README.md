1.项目介绍：
  go初学者、编程小白练手
  BMI 健康评测工具 - 一个基于终端的交互式程序，用于计算身体质量指数（BMI）并提供健康评级。
2.功能特性
  通过终端交互式输入身高和体重。
  自动计算 BMI 值。
  根据国际通用的 BMI 标准提供健康评级（偏瘦、正常、偏胖、肥胖）。
  支持连续多次评测，并可选择退出。
  内置了基本的输入错误处理（如非数字、负数输入）。
3.如何开始
  需要安装Go语言环境， Go1.16+
  git clone https://github.com/Antitelescope/BMIChecker
  cd BMIChecker
  go run main.go
4.项目结构与设计思路
  输入接口 - 终端输入器
  输出接口 - 终端输出器
  核心逻辑 - BMI计算和评级逻辑
  BMI测量结构体
  main函数
5.未来扩展计划
  当前版本聚焦于核心逻辑的实现。为了未来能方便地扩展（如添加 Web 界面或数据库存储），后续计划通过引入 接口（Interface） 来重构输入和输出模块，将核心计算逻辑与具体的交互方式解耦。遵循 面向接口编程 的原则，使得程序能够轻松替换或增加新的输入源（如 Web 表单、文件）和输出目标
